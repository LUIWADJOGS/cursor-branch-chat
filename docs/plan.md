# План работ

## Цель

Сделать `cursor-branch-chat` более удобным, надежным и готовым к дальнейшему развитию.

## Этапы

### 1. Базовая стабилизация

- добавить `README` с описанием проекта, запуском и сценариями использования
- обработать ошибки: нет workspace, нет git-репозитория, branch не определился, deeplink не открылся
- привести сообщения пользователю к единому стилю

### 2. Улучшение UX

- добавить сортировку чатов по дате обновления или создания
- добавить просмотр архивных записей
- добавить действия `rename`, `unarchive`, `delete`
- улучшить отображение записей в панели: статус, дата, контекст

### 3. Улучшение prompt-логики

- вынести шаблон prompt из кода в настройки extension
- дать возможность менять шаблон под проект или под ветку
- продумать безопасную обработку длинных prompt

### 4. Надежность и поддерживаемость

- покрыть тестами `chatRegistry`
- покрыть тестами `buildBranchPrompt`
- покрыть тестами `createPromptDeeplink`
- проверить edge cases для `getCurrentBranch`

### 5. Расширение возможностей

- поддержать несколько workspace folders
- добавить поиск и фильтрацию по чатам
- продумать экспорт и импорт записей

## Приоритет

1. документация и базовая прозрачность проекта
2. обработка ошибок и стабилизация UX
3. настройка шаблона prompt
4. сортировка и работа с архивом
5. тесты для ключевой логики
6. расширенные функции

## Ближайшие задачи

1. улучшить `README.md`
2. вынести шаблон prompt в настройки расширения
3. добавить отображение архивных чатов
