## Что уже понятно по проекту

- это расширение для `Cursor` / `VS Code`
- проект написан на `TypeScript`
- сборка идет через `tsc`
- данные хранятся в `context.globalState`
- фильтрация чатов идет по текущей `git`-ветке
- открытие существующих чатов работает через internal command Cursor `composer.openComposer`
