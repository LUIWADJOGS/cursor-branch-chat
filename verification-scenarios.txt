# Branch Chat Extension — сценарии проверки

## Deeplink
1. Выполнить "Create Branch Chat", ввести title.
2. Должен открыться новый чат Cursor с префиллом: "Ветка: <текущая ветка>", "Задача: <title>", правила работы.
3. URL в браузере/приложении должен быть вида cursor://anysphere.cursor-deeplink/prompt?text=...

## Хранение
1. Создать branch chat на ветке feature/x.
2. Перезапустить Cursor.
3. Открыть панель "Branch Chats" (иконка чата в Activity Bar) — запись для feature/x должна отображаться.
4. Кликнуть по записи в "Branch Chats" — должен открыться существующий чат Cursor, а не создаться новый.
5. Архивировать запись через контекстное меню — она исчезнет из списка для текущей ветки.

## Фильтрация по ветке
1. На ветке feature/x создать 1–2 branch chat.
2. Переключиться на ветку main (или другую).
3. В панели "Branch Chats" должны отображаться только чаты для main (или пустой список).
4. Вернуться на feature/x — снова видны чаты для feature/x.
5. При переключении ветки (если есть чаты для новой ветки) — опциональное уведомление: "Branch: <name>. Chats for this branch: N."

## Команды
- "Show Chats For Current Branch" — Quick Pick со списком чатов текущей ветки; выбор открывает существующий чат Cursor.
- Контекстное меню по записи в дереве: Open in Cursor Chat, Copy Prompt, Archive.
